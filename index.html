About me

I'm a highly motivated and quick-learning junior data scientist with a passion for transforming raw data into actionable insights. My journey into the world of data science began with a strong foundation in machine learning, natural language processing, and data visualization. I'm particularly drawn to the power of data to solve real-world problems and drive informed decision-making.

During my recent 3-month internship at Innomatics Research Labs, I had the incredible opportunity to apply my skills to a variety of challenging projects. I developed a RAG (Retrieval-Augmented Generation) system using LangChain, enabling more comprehensive and context-aware information retrieval from company-specific data sources. I also built a sentiment analysis solution for Flipkart product reviews, leveraging machine learning techniques and deploying it with MLflow for seamless model management.

These experiences solidified my passion for data science and fueled my desire to continue learning and growing in this dynamic field. I'm eager to contribute my skills and enthusiasm to a team of data professionals where I can make a meaningful impact.

    Innomatics Research Labs

    Data Science Intern

    During my internship at Innomatics Research Labs, I gained invaluable experience in applying data science principles to real-world challenges. I worked on a variety of projects that spanned machine learning, natural language processing, and data engineering.

    Developed a RAG (Retrieval-Augmented Generation) System
    Leveraging LangChain, I built a system that integrated large language models (LLMs) like Gemini 1.5 Pro with company-specific data sources. This enhanced information retrieval and generation capabilities, enabling more comprehensive and context-aware responses.

    Built a Sentiment Analysis Solution
    I designed and implemented a sentiment analysis solution for Flipkart product reviews using machine learning techniques. This involved data preprocessing, feature extraction, model training, and evaluation. I utilized MLflow for efficient model management and Prefect Workflow for automated execution.

    Enhanced Video Subtitle Search Engine
    Development of an advanced search engine algorithm specifically designed for video subtitles. By employing natural language processing and machine learning techniques, I significantly improved search relevance and accessibility, making it easier for users to find the information they need.

    Conducted Exploratory Data Analysis (EDA)
    I performed in-depth EDA on the AMCAT dataset, investigating relationships between various features and the target variable, Salary. This involved statistical analysis, hypothesis testing, and data visualization to uncover insights and validate claims regarding earning potential in different fields.

    Developed and Deployed a Regex Matching Web Application
    I created a user-friendly web application using Flask and HTML that replicated the functionality of regex101.com. Users could input test strings and regular expressions to perform pattern matching. I deployed the application on AWS EC2 for scalability and accessibility.
